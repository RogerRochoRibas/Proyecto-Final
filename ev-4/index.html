Lenguaje: HTML

El contenido de la página. Los párrafos, títulos, secciones, imágenes, videos, menús, botones, etc. La base de todo. Es muy importante elaborar el código HTML de forma limpia y ordenada, todo lo que vendrá después, así como el mantenimiento posterior de la página dependerán de ello.

<h1>Soy un título</h1>

<p>Soy un párrafo</p>

<a>Soy un enlace</a>

Lenguaje: CSS

El CSS define los estilos de la página. Colores, tipografias, espaciados, colocación y ordenación, tamaños. Las normas CSS funcionan con un selector, que define los elementos que se verán afectados, y las propias normas.

p {color:red} Establece en rojo el color de los párrafos (p).

h1 {font-family:Arial} Establece la tipografía de los títulos (h1) como Arial.

a {margin: 15px} Establece 15 píxeles de margen en los enlaces (a).

Lenguaje: JavaScript

Nos permite crear una web interactiva, por ejemplo, modificando los elementos de la página al pulsar un botón. Las funciones de JavaScript deben iniciarse, por ejemplo, utilizando onLoad se iniciará la función elegida cuando se cargue un elemento, y onClick, cuando hagamos un click en el elemento.

function myFunction() {

var x = document.activeElement.tagName;

document.getElementById("demo").innerHTML = x; }

En este ejemplo creamos una variable con el "tagName" y lo introducimos en el elemento identificado como "demo".

¿En qué consiste el curso?

Competencia general

Crear y publicar páginas web que integren textos, imágenes y otros elementos, utilizando lenguajes de marcas y editores apropiados, según especificaciones y condiciones de «usabilidad» dadas y realizar los procedimientos de instalación y verificación de las mismas en el servidor correspondiente.

Módulos Formativos

Horas lectivas totales: 480H

Módulo de prácticas profesionales: 80H

Crear los contenidos en HTML

Horas: 80

Aprender las etiquetas HTML, desde los <p> hasta los <iframe> pasando por las listas con sus <ul> y sus <ol> llenos de <li>, los <img> y los <video> que enlazan archivos, así como los <span> que ni se ven, los <div> que organizan la página junto a los <section> o los <aside>.

Hojas de estilos CSS

Horas: 58

Aprendemos a modificar el aspecto de la página. Desde darle color a un texto o cambiar la tipografia, pasando por colocar los elementos, darles márgenes para que no esten pegados o alinearlos. También podemos hacer que cambien según el tamaño de la pantalla, todo lo necesario para que la página se vea como queremos nos lo permite el CSS.

Plantillas y formularios

Horas: 48

Utilizar plantillas para crear páginas web, se trata de una forma rápida de obtener buenos resultados, pero límita el margen de maniobra del diseñador.

La creación de formularios con muchos tipos de <input> donde se introduce la información, todos con sus <label> para que el usuario sepa lo que debe introducir, entre otros elementos.

Programación con JavaScript

Horas: 72

El JavaScript le da comportamiento a la página. Permite que el usuario interactue con la página, y que la página actue por si misma bajo ciertas condiciones. Por ejemplo, permite modificar el contenido de la página bajo ciertas condiciones o cuando el usuario pulsa un botón.

Validación de datos y efectos especiales

Horas: 72

Validar los datos que se introducen en un formulario, que en el campo Email nos introduzcan un email valido, y que en caso contrario se informe al usuario.

Trabajar con selección de imagenes, creación de textos mejorados y efectos estéticos, trabajar con marcos y ventanas multiples.

Publicar páginas web

Horas: 66

Para que una página sea accesible para el público básicamente necesita dos cosas, un dominio, como www.espaivisual.cat y un servidor que aloje la página.

El primero es una dirección que debemos comprar y renovar anualmente, el segundo es un servicio de un tercero, que guardará nuestra web y la servirá a quien acceda a la dirección.
